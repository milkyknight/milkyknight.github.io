HDU - 6704 后缀数组 + 主席树
2019-09-12

给一个字符串 s
对于每个询问 q(l,r,k) 返回子串 sl-r 第k次出现的位置
由于后缀数组按照字典序排序，所以可以主席树维护SA，这样就能找到第k次出现的位置
取 l 为左端点的后缀，len = r - l + 1,那么考虑排名左右的后缀中 lcp(i,i+x) >= len,
lcp(i,i-x) >= len 的后缀一定存在相同子串，由于后缀数组的性质，可以考虑二分找到 x
